## ⚡ Начало работы

0️⃣ Загрузите <a href="https://github.com/usebruno/bruno/releases">клиент Bruno</a> для работы с API

1️⃣ Найдите список коллекций → нажмите `+` → выберите **Import collection**

2️⃣ Выберите источник **Git Repository** → в поле ввода вставьте ссылку на репозиторий:

`https://github.com/thrustcq/api_yandex360.git`

3️⃣ При необходимости укажите локальную папку для сохранения коллекции, затем нажмите **Clone**

4️⃣ После успешного копирования выберите новую коллекцию **api_yandex360** и нажмите **Open**

5️⃣ Настройте окружение, в правом верхнем углу экрана найдите раздел **Environments** и перейдите в настройки, добавьте токен авторизации для нужной организации и остальные требуемые параметры

✅ Готово!
